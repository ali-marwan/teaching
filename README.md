# Module 20: Hydrocarbons

Official classroom HTML presentation app for Grade 11 Advanced Chemistry.

## Contents

- Multi-lesson tabbed presentation structure for Module 20.
- Complete Lesson 1: Introduction to Hydrocarbons.
- Local assets and local Three.js runtime for offline classroom use.
- PDF-safe print styling with one slide per page and static 3D fallbacks.

Open `index.html` in Chrome to present or export to PDF.

## Media

- Teacher-provided PowerPoint media extracted from the local lesson files.
- Ruwais refinery image from the existing local deck assets.
